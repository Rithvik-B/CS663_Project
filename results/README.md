# Results Directory

This directory contains sample results from PCA-Mixed NST experiments:

- `metrics_summary.csv`: Metrics for generated images
- Grid visualizations showing alpha interpolation
- Sample stylized images

## Expected Structure

After running experiments, you should have:

```
results/
├── metrics_summary.csv          # Combined metrics for all experiments
├── <content>_<style1>_<style2>_<method>_grid.jpg  # Grid visualizations
└── individual_outputs/            # Individual stylized images
```
